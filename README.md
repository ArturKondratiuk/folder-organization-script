# Folder Organization Script

## Project Description

File Organizer CLI is a Java console application that allows users to navigate directories and manage files directly from the terminal.

The program can automatically sort files into folders based on their type.

---

## Technologies Used

- **Java**
- Standard Java libraries:
  - `java.io.File`
  - `java.nio.file`
  - `java.util`

- **Batch**
  - `compiler.bat` - to compile the file without CLI

---

## Setup Instructions (2 ways)

### 1st Way:
  - `Downoload zip and extract the files`
  - `Go folder-organization-script\src`
  - `Run the compiler.bat`
  - `follow instructions`


### 2nd Way
  - `git clone https://github.com/ArturKondratiuk/folder-organization-script.git`
  - `cd folder-organization-script`
  - `cd src`
  - `java App.java`
  - `follow instructions`
